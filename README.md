# Dis Thing Dat Thing with CCC
***Dis Thing Dat Thing with CCC*** is an educational interactive that aims to train memorization of DSE Biology vocabulary through helpful menus and quizzes.

The game was a school-based assignment for one of my elective subjects, and is programmed in Python in ONE script, heavily utilizing the PyGame library.

This repository serves as an archive of my work and will not be maintained and rewritten in the foreseeable future.

## How To Run
This program requires Python 3.13.x and the PyGame 2.6.1 library to run. The
instructions to run the program are as follows:
0. Install Python 3.13.x. 
1. Download the source code as a .zip file.
2. Extract it.
3. Run an IDE to import the folder as a project.
4. Configure your Python interpreter, set the Python version to 3.13.x.
5. Install the PyGame library. Make sure the version is 2.6.1.
6. Run `game.py`. Do NOT isolate the `assets/` folder in the root folder as it is
ESSENTIAL to the program.

## Instructions
This program makes use of the mouse and the keyboard. Dictionary Mode, Quiz Mode and Credits can be accessed via the menu selection in the Main Menu, with the use of clickable buttons.

In the presence of I-Beams (or insert underscores), a keyboard may be used to input characters.

To submit/confirm the input, press ENTER/RETURN on your keyboard.

If applicable, use the mouse wheel to scroll up/down menus.

To exit a menu, press ESCAPE on your keyboard.

For a more in-depth explanation of the game's controls and operation, click on
the "Manual" button in the Main Menu.
