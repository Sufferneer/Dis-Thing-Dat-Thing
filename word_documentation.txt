This text file shows the format of words in all dictionary entries.

Rules:
- All the wording of keywords abides Biology DSE guidelines.
- All keywords are either nouns or noun phrases
- All words uses their British spelling by default
- Should the British spelling be different to the American spelling,
    It should be supplemented in the "alt_spellings" array.
- Translations should be in Traditional Chinese.
- All Chinese characters should use the most widely accepted writing forms.
(e.g. 睾 should be 睪)
- If possible, definitions and translations should abide by official Biology DSE definitions.
- Some Chinese translations of molecules with long names (e.g. DNA, ATP, etc.) are
not used in Chinese DSE. Avoid including them.

Variables:
"word": String. The keyword's default wording
"plural": String. The keyword at its plural form. Leave empty if none. Copy the "word" variable if
the keyword's plural form is the same as the singular one.
"full": String.
"redirects": Array<String>. Lists of words that shows the keyword's default wording when searched.
Useful for synonyms. Leave empty if no synonyms exist.
"alt_spellings": Array<String>. List of acceptable spellings in the DSE exam.
Leave empty if none exists.
"forms": Dictionary<Array<String>>. The keyword's forms in different parts of speech.
Leave empty if none is commonly used by major publishing documents.
"word_class": The part of speech of the keyword.
"definition": String. The definition of the keyword.
"translation": String. The Chinese translation of the keyword.
Make the same as "word" if Chinese translation is the same or does not exist.
